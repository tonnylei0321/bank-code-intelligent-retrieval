# 前端菜单测试指南

## 🔄 重启前端

### 1. 停止前端服务
```bash
# 找到前端进程
ps aux | grep "vite" | grep -v grep

# 停止进程（替换 PID）
kill <PID>
```

### 2. 启动前端服务
```bash
cd frontend
npm run dev
```

## ⚠️ 常见问题

### Q1: 菜单没有更新？
**解决方案**:
1. 硬刷新浏览器 (Cmd+Shift+R)
2. 清除浏览器缓存
3. 使用无痕模式访问
4. 检查前端是否重启成功

### Q2: 点击菜单没反应？
**解决方案**:
1. 检查浏览器控制台是否有错误
2. 确认路由配置正确
3. 确认前端服务正常运行

### Q3: 页面显示404？
**解决方案**:
1. 检查 App.tsx 中的路由配置
2. 确认组件导入路径正确
3. 重启前端服务
